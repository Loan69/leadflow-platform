Guide — Trigger IMAP n8n

Surveillance de boite mail avec n8n

Le nœud Email Trigger (IMAP) de n8n interroge une boite mail à intervalle régulier et déclenche le workflow dès qu'un email non lu est détecté. Il marque automatiquement l'email comme lu pour éviter tout retraitement.

Pour le workflow LeadFlow, configurer ce trigger sur la boite mail qui reçoit les demandes de contact de tes clients (ex : contact@agence.com).

Procédures détaillées Cas nécessitant un mot de passe d'application

📧 Gmail — Créer un mot de passe d'application

1
Aller sur myaccount.google.com → Sécurité
2
Vérifier que la Validation en 2 étapes est activée (obligatoire pour la suite)
3
Dans la barre de recherche du compte Google, taper mots de passe des applications et ouvrir le résultat
4
Saisir un nom (ex : n8n LeadFlow), cliquer Créer
5
Copier le mot de passe de 16 caractères généré → le coller dans le champ Password du credential IMAP n8n
6
Dans Gmail → Paramètres → Voir tous les paramètres → Transfert et POP/IMAP → vérifier que IMAP est activé

📘 Microsoft 365 / Outlook.com — Créer un mot de passe d'application

1
Aller sur account.microsoft.com → Sécurité → Options de sécurité avancées
2
Activer la Vérification en deux étapes si ce n'est pas fait
3
Dans la section Mots de passe d'application, cliquer Créer un nouveau mot de passe d'application
4
Copier le mot de passe généré → champ Password dans n8n
5
Dans Outlook Web → Paramètres → Courrier → Synchronisation → POP et IMAP → activer IMAP si nécessaire

🟣 Yahoo — Créer un mot de passe d'application

1
Aller sur login.yahoo.com → icône compte → Sécurité du compte
2
Activer la Validation en deux étapes
3
Cliquer Générer le mot de passe d'application → nommer-le n8n
4
Copier le mot de passe → champ Password dans n8n
Conseils & résolution de problèmes
📁
Surveiller un sous-dossier
Dans le nœud n8n, le champ Mailbox accepte n'importe quel chemin : INBOX/Leads. Utile pour filtrer les emails via une règle de messagerie avant que n8n ne les traite.
⏱️
Fréquence de polling
Par défaut configuré à 5 minutes dans le workflow LeadFlow. Réduire à 1 minute pour les leads urgents, ou augmenter à 15 minutes pour économiser les exécutions (plan gratuit n8n limité).
🔐
Erreur certificat TLS
Si n8n retourne une erreur UNABLE_TO_VERIFY_LEAF_SIGNATURE, activer Allow Unauthorized Certs dans le credential. Courant sur les serveurs OVH mutualisés.
🚫
Authentification refusée
Pour Gmail, Microsoft et Yahoo : le mot de passe du compte ne fonctionnera pas — il faut impérativement un mot de passe d'application dédié à n8n.
📬
Boite dédiée recommandée
Créer une adresse spécifique (ex : leads@agence.com) et configurer une règle de redirection depuis la boite principale. Évite les faux positifs et les emails personnels traités par erreur.
✅
Tester la connexion
Dans n8n, après avoir rempli le credential, cliquer "Test" (bouton dans la modal). Un message vert "Connection tested successfully" confirme que tout est correct avant d'activer le workflow.
